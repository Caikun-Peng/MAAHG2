# MAAHG2
**MAA Houkai Gakuen 2**

基于 [MaaFramework](https://github.com/MaaXYZ/MaaFramework) 的《崩坏学园2》自动化程序。  
目前正在开发中.....
</div>

## 目标功能
- [X] 打开游戏(从任意位置)
- [X] 从上次登录进入游戏
- [X] 收取邮件
- [X] 领取社团奖励
- [X] 购买零时之种
- [X] 多元裂缝快捷战斗
  - [X] 虚轴之庭
  - [X] 使魔的爱
- [X] 使魔探险
- [X] 领取每周任务奖励
- [X] 领取每日存在感奖励
- [ ] 崩科校活
    - [X] 实验楼领取使魔碎片
    - [ ] 占卜
    - [ ] 战斗
- [ ] 无自动套挂机
    - [ ] root设备
        - [X] python实现
        - [X] 整合进MAA
        - [ ] 
    - [ ] 非root设备
