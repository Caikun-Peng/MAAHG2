## 目标功能
- [X] 打开游戏(从任意位置)

- [ ] 收取邮件
- [ ] 领取社团奖励
- [ ] 购买零时之种
- [ ] 崩科校活
- [ ] 使魔探险
- [ ] 领取每周任务奖励
- [ ] 领取每日存在感奖励
- [ ] 领取任务奖励
- [ ] 挂机打关
